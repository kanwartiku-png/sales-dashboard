# Sales Performance Analytics Dashboard

## Project Overview

Interactive Power BI dashboard developed to analyze sales performance, profitability, customer behavior, and regional business trends.

## Business Objective

Provide business leaders with actionable insights to improve revenue growth and profitability.

## Tools Used

- Power BI
- Excel
- DAX
- Data Modeling

## Key KPIs

- Total Sales
- Total Profit
- Profit Margin
- Quantity Sold
- Regional Performance
- Category Performance

## Dashboard Features

- Sales Trend Analysis
- Regional Performance Tracking
- Category Analysis
- Profitability Monitoring
- Customer Segment Insights

## Key Findings

- Technology category generated highest sales.
- West region contributed the largest share of revenue.
- Discount-heavy transactions reduced profitability.

## Dataset

9,994 transaction records.
